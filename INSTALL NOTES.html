1. Creating a group :
    • Use the "New" button or the right-tree context menu (select an empty place).
  2. Renaming a group :
    • Use the right-tree context menu
    or
    • Clic the Group to access editing
    or
    • Change the name in the "Group" section, then propagate with the button ">>".
  3. Add a Service to a Group
    • Use the left-tree context menu (a group must be selected on the right).
    or
    • Select a service, then use the "Add" button (a group must be selected on the right).
    or
    • Drag and Drop the wanted Service from the left-tree into a Group on the right-tree.
  4. Removing a Service from a Group
    • Select the desired Service under the desired Group
    then
    • Use the "Remove" button.
    or
    • Use the right-tree context menu.
  5. What is "Latest" ?
    • You can flag one or more Service(s) inside a Group as "Latest".
    • They will then start only AFTER THE OTHERS and be stopped AT FIRST.
  6. Important Note :
    • Groups are identified BY NAME, which must be UNIQUE.
    • Even if you can create multiple groups using the same name, only one of them will "survive" the restarting of the tool.
